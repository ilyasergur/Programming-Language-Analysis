# Programming-Language-Analysis

Files Included
● File_IO.java: Source code for the Java implementation.
● File_IO.cs: Source code for the C# implementation.

● File_IO-datas.xlsx: Detailed benchmarking data including runtime, memory, and CPU
metrics for 20+ test iterations.

Findings
Based on the results obtained during the tests conducted
● Compilation Time: The compilation time of Java was more than that of C#.
● Runtime Efficiency: There was parity between the runtime efficiency of the two
languages, with the runtime efficiency of C# being slightly better in terms of
runtime latency in this particular case.
● Memory Consumption: The memory consumption by both systems was measured,
and differences were noted in the garbage collection mechanism of both.
